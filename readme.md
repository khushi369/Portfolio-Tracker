# 📈 Portfolio-Tracker
A full-stack web application that mimics a stock trading platform. Users can view holdings, track positions, and place buy/sell orders.

## ✨ Features
- View all stock holdings with quantity, average price, and profit/loss.
- See open positions.
- Place new orders (buy/sell).
- (Coming soon: live stock prices, user authentication, portfolio charts)

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Other:** Mongoose, CORS, DotEnv
